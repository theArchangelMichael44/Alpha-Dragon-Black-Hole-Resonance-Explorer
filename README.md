# Alpha Dragon: Black Hole Resonance Explorer

An interactive, browser-based simulation exploring how oscillating attraction fields produce **shell-like structures, resonance bands, and pulse behavior** during collapse.

This system combines inward pull, oscillation, and stochastic variation to generate dynamic, layered structures that resemble black hole shell formation and field resonance patterns.

---

## 🧠 Overview

Particles are influenced by:
- a central attractor (gravity-like pull)
- distance-based oscillation (resonance behavior)
- controlled randomness (noise)

As the system evolves, particles form visible **density bands and shell-like structures**, driven by the interaction between inward collapse and oscillatory forces.

These structures are not static — they pulse, shift, and reorganize continuously.

---

## 🧪 What This Demonstrates

- Resonant collapse behavior
- Shell and band formation under oscillation
- Density clustering and fluctuation
- Pulse-like behavior emerging from simple rules
- Field reorganization around a moving center

---

## 🎮 Controls

- **Click / Drag** → Move center  
- **SPACE** → Pause / Resume  
- **R** → Reset simulation  
- **M** → Switch mode (Stable / Resonant / Chaotic)  
- **T** → Toggle trails  
- **↑ / ↓** → Adjust pull strength  
- **← / →** → Adjust oscillation strength  
- **N** → Adjust noise level  
- **P** → Add particles  
- **O** → Remove particles  

---

## ⚙️ How to Run

1. Download or clone the repository  
2. Open the `.html` file in any modern browser  
3. No installation required  

Runs entirely client-side.

---

## ⚙️ How to Experiment

You can modify behavior directly in the code:

- Adjust particle count
- Tune pull strength and oscillation
- Modify noise levels
- Explore different mode behaviors

A central `CONFIG` block is provided for easy tuning.

## 🧩 System Notes

- Collapse is softened to avoid singularities  
- Oscillation introduces resonance bands  
- Noise creates variation and instability  
- Shells emerge from competing forces, not predefined structures  

“Shells” here refer to **emergent density layers**, not fixed physical boundaries.
